HTML

  • The head element contains information, or metadata, about the webpage.
  • The title element controls what shows up in the browser tab for the website.
  • The body element represents the visual content shown to the user.
  • The section element organizes the content on the webpage by topic.
  • The name of a semantic element describes its content and role on the webpage.
  • Alt text for images is important for accessibility and can improve SEO

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS is used to style an element directly in the HTML. For a particular page element only, and not very flexible.
  • Internal style sheet: contains CSS rules for the webpage in the head section of the HTML. The rules apply only to that page.
  • External style sheet: preferred bc it has all CSS rules in a separate file, which makes changes easier. Linked using a link element in the head of the file.
  • The * selector (wild card) applies the rules to all elements on the page.
  • Using the class attribute as a selector allows you to choose which elements will be assigned to CSS rule.

Git

  • git status: Checks what branch we are currently on
  • git checkout branch-name: Switches from current to named branch
  • git checkout -b branch-name: Creates a new branch and switches to it
  • git pull origin main: Pulls down current version of main branch
  • git add -A: Adds the changed files to the staging area of git
  • git commit -m "message": Commits the changes alongside a message about what was done
  • git push origin branch-name: Pushes up the changes to the named branch
  • Git flow = add, commit, pull, push. Pull request, merge, close issue.

JavaScript

  • A variable is a named container that allows us to store data in the code. The = assigns data to a variable.
  • A method is a set of instructions that can be executed by a computer, ex: .log()
  • An argument is the data inside of the parenthesis of a method.
  • Link a script.js file to the index.html file right above the closing tag for the body element.
  • Control flow is the order in which a computer executes code in a script. In JavaScript, code runs in order from top to bottom.
  • Conditional statements and loops can interrupt control flow and decide the order in which code is executed.
  • When using an if statement, code will only execute if the statement meets a condition or is true.
  • Falsy values: 0, -0, On, “”, null, undefined, NaN, or a boolean with the value of false.
  • All other values are considered truthy
  • An array is a single variable used to hold a group of related data.
  • An index is a unique number used to identify the location of a piece of data in an array. The first item in any array has an index of 0.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array. It allows a single code block to be executed repeatedly.
  • A function is a set of instructions that tells the computer how to perform a certain task. It must be called in order to execute.
